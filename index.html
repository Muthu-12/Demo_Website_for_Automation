What is Selenium WebDriver? ▼
Selenium WebDriver is a browser automation framework that allows you to write tests in multiple programming languages including Java, Python, and C#.
What is the difference between findElement and findElements? ▼
findElement returns the first matching element and throws NoSuchElementException if not found. findElements returns a list of all matching elements, returning an empty list if none found.
What are the types of Waits in Selenium? ▼
There are three types: Implicit Wait (applied globally), Explicit Wait (waits for a specific condition), and Fluent Wait (polls at a defined frequency with custom timeout).
How to handle iFrames in Selenium? ▼
Use driver.switchTo().frame(id/name/index/element) to switch into an iframe. Use driver.switchTo().defaultContent() to return to the main page, and driver.switchTo().parentFrame() to go to the parent frame.
