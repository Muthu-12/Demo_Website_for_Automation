16 Accordion / Collapsible id="accordion"
What is Selenium WebDriver? ▼
Selenium WebDriver is a browser automation framework that allows you to write tests in multiple programming languages including Java, Python, and C#. It directly communicates with the browser.
Difference between findElement and findElements? ▼
findElement returns the first matching element and throws NoSuchElementException if not found. findElements returns a list of all matching elements — empty list if none found.
Types of Waits in Selenium? ▼
Three types: Implicit Wait (applied globally), Explicit Wait (waits for a specific condition using WebDriverWait), and Fluent Wait (polls at a defined frequency with custom timeout).
How to handle iFrames in Selenium? ▼
Use driver.switchTo().frame(id/name/index/WebElement) to switch into an iframe. Use driver.switchTo().defaultContent() to return to the main page.
